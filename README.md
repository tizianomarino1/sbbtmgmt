# sbbtmgmt

Manager per suite di black box test di InfoCamere.

## Avvio progetto Swing con Maven

Prerequisiti:
- Temurin JDK 25 installato
- Maven 3.9+

Comandi utili:

```bash
mvn clean compile
mvn exec:java
```

L'applicazione apre una finestra Swing con due campi:
- nome fornitore
- partita IVA
